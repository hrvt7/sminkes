# Makeup by Kamilla Kovacsik — Sminkes weboldal

Professzionális sminkes weboldal Kovacsik Kamilla makeup artist számára. Győr & Tapolca.

## Stack
- Next.js 15 (App Router)
- TypeScript
- Tailwind CSS 4
- Vercel Analytics + Speed Insights

## Fejlesztés

```bash
npm install
npm run dev   # http://localhost:3000
npm run build
```

## Struktúra

- `src/app/` — oldalak (home, szolgáltatások, legal)
- `src/components/` — UI komponensek (Hero, Services, Schedule, stb.)
- `public/` — robots.txt, sitemap.xml, llms.txt, képek

## Deploy

Automatikus Vercel deploy a `main` branchről. Preview branch-ekre preview URL.
